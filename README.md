# 📊 KPI Scorecard Generator

## 📝 Overview

The KPI Scorecard Generator is a powerful, client-side web application designed to automate the creation of beautiful, highly visual performance scorecards for Customer Support, Moderation, and QA teams.

Built as a self-contained, zero-build-step tool, it allows Team Leads and Managers to effortlessly input agent metrics, define custom KPIs, and export professional-grade graphics (individually or in batch ZIP files) to share during performance reviews.

Designed to be highly portable, the entire application is contained within a single file (HTML/CSS/JS), making it fully compatible with embedded environments like WordPress Custom HTML blocks, iFrames, and standalone local execution.

# 🛠️ Tech Stack

## Core Technologies

### HTML5
Semantic structure utilizing `<template>` tags for efficient DOM cloning of repetitive elements (agents, KPIs).

### CSS3

- Advanced use of CSS Variables (Custom Properties) for instant theme switching and maintainable palettes
- CSS Grid & Flexbox for a fully responsive layout
- Glassmorphism styling using `backdrop-filter` and transparent background layers

No frontend frameworks (React/Vue/Angular) used.

- Maximum portability
- Zero compilation requirements

# 📦 Libraries & How They Are Used

## html2canvas

**Usage:**  
Captures the dynamically generated HTML scorecard DOM element and converts it into a high-resolution base64 PNG image.

---

## SheetJS (`xlsx`)

**Usage:**  

- Parses uploaded `.xlsx` files into JavaScript objects for the Smart Import feature
- Builds 2D arrays from the active DOM state to export custom Excel templates

---

## JSZip

**Usage:**  

Used during the Batch Export process:

- Creates an in-memory ZIP archive
- Injects base64 images generated by `html2canvas`
- Triggers a single compressed file download

---

## Chart.js & datalabels plugin

**Usage:**  

Renders mini bar charts inside the scorecard during `2-Month Comparison` mode.

Data labels print exact metric values directly above the bars.

---

## Font Awesome 6 & Flag Icons

**Usage:**  

Provides scalable vector icons for:

- UI buttons
- KPI visual indicators
- Language selection flags

# 🧠 Advanced Technical Implementations

## Event Delegation

Instead of attaching hundreds of event listeners to dynamically created elements (such as `Delete Agent` buttons), the app uses a single event listener at the document level.

### Benefits

- Optimized memory usage
- Better scalability
- Cleaner event management

---

## Asynchronous Rendering Loop

The Batch ZIP export function uses:

- `async/await`
- `Promises`
- Simulated delays with `setTimeout`

This ensures the DOM has enough time to fully update and render charts before `html2canvas` captures snapshots.

### Prevents

- Blank images
- Overlapping renders
- Partial chart rendering

---

## DOM `<template>` Cloning

Instead of generating massive HTML strings dynamically, the app relies on hidden `<template>` tags.

JavaScript simply:

1. Clones the template node
2. Injects the required data
3. Renders the final component

### Benefits

- Cleaner architecture
- Better maintainability
- Modular UI generation

---

## Custom Modals vs. Native Dialogues

Native browser popups like:

- `alert()`
- `confirm()`

are frequently blocked inside sandboxed iframes or CMS environments.

The application uses a custom CSS-driven modal engine with callback functions to ensure full compatibility across platforms like WordPress.

# 🚀 How to Use (Workflow)

## 1. Headers & Mode

Define:

- Reporting period (e.g., February vs. March)
- Mode (`Single` or `Comparison`)
- Number formatting (`Abbreviated` or `Full`)

---

## 2. Team Achievements

Add global wins displayed on every agent scorecard.

### Example
- Hit 95% Global CSAT

---

## 3. KPIs

Add metrics and configure:

- Icons
- Pass thresholds
- Fail thresholds

This automatically powers the color-coding system on the final scorecards.

---

## 4. Feedback Defaults

Set a default motivational quote using:

```text
{name}
```

as a dynamic variable.

---

## 5. Agents Data

Add agents manually or use the Excel Import/Export system:

1. Download template
2. Fill it in Excel
3. Upload it back into the application

---

## 6. Generate

Review the generated scorecards in the workspace and use the download icons to save them locally.
