KPI Scorecard Generator

How to Use the Generator

Welcome! This tool helps you generate beautiful performance cards for your team. Here's a quick guide to getting started:

1 Headers & Mode

Configure the basic structure of your report. You can choose between two main modes:

  • 2 Months Comparison: Ideal for showing progress. It compares the previous month with the current one, automatically calculating trends (▲▼). In this mode, you can also enable Bar Charts for individual KPIs.
  • Single Month Report: A cleaner view focusing only on the current month's performance. The data is centered and simplified.

You can also choose how numbers are formatted (e.g., "1.5k" vs "1,500"). The system auto-generates the month names based on the current date, but you can overwrite them.

2 Team Achievements

Highlight global metrics or wins that apply to the whole team. These will appear consistently on every agent's card. The section title is fully editable.

3 Define KPIs (Metrics)

This is where you define the metrics you want to track. You can add them manually or use the presets.

  • Role Presets: Click on a role (e.g., "CS Agent") to instantly load a standard set of KPIs and a dummy agent.
  • Custom Configurations: Use the Save to Browser button to store your complex KPI setups for future use, or Export/Import to share your KPI structures with other managers.
  • Goals & Colors: Set a Goal (Green) and a Red Below threshold. The system will automatically color-code the results based on these targets.
  • Volume Metrics: If a metric is just a count (like "Total Tickets") and doesn't have a pass/fail goal, simply leave the Goal and Red thresholds blank. It will be displayed in a neutral color.
  • Charts: If you are in Comparison Mode, you can change a KPI's "Chart Type" to a Bar Chart for visual emphasis. Note: This option is automatically ignored in Single Month mode to keep the design clean.

4 Feedback Defaults

Set a default motivational quote that will appear at the bottom of the cards. You can use the placeholder {name} to automatically inject the agent's first name.

5 Agents Data

Add the actual performance data for your team. You can do this manually, but the fastest way is using Excel:

  • Download Template: First, define your KPIs in Step 3. Then, download the template. It will be custom-generated based on your KPIs and Mode.
  • Smart Import: Fill in the Excel file and upload it. The system will automatically map the data to your agents.
  • Auto-KPI Creation: If you upload an Excel file with metrics that don't exist in Step 3 yet, the system is smart enough to automatically create those missing KPIs for you based on the column headers!
